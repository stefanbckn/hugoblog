POSSE staat voor Publish (on your) Own Site, Syndicate Elsewhere.

De bedoeling is heel eenvoudig. Ik post alles op mijn eigen blog onder [notes](/notes/) en van daaruit gaat het automatisch verder naar andere social media platformen via RSS (Really Simple Syndication).

### Waarom kies ik voor POSSE?

Er zijn verschillende redenen om voor POSSE te kiezen. Bij mij komt het neer op twee dingen. Eigenaarschap van mijn content en minder afhankelijk zijn van een derde partij. Als ~~twitter~~, X, bluesky, mastodon,... verdwijnt dan heb ik alles nog hier op mijn blog.

### Hoe post ik dan via mijn smartphone?

iOS shortcuts!

Ik heb een shortcut gemaakt die op de volgende manier werkt:
- Ik schrijf mijn tekst via “Ask for text”.
- Een titel geef ik ook mee via “Ask for text”. Spaties worden automatisch vervangen door een dash (‘-‘).
- Via een “Choose from menu” kan ik kiezen om foto's toe te voegen.
  - Indien ja: 1 of meerdere foto’s kunnen geselecteerd worden.
  - Foto wordt omgezet naar jpeg en via Working Copy in mijn repo gezet.
  - Per foto wordt alt-tekst opgevraagd via "Ask for text".
  - De alt-tekst wordt samen met de nodige markdown tags toegevoegd aan de post.
- Als laatste wordt alle tekst samengevoegd tot een post die klaar is voor mijn blog.

Hierna zorgt Working Copy voor de magie om alles in mijn repo te posten (pull - commit - push). De website wordt via GitHub en Netlify gebouwd, de RSS feed wordt gepubliceerd en Micro.blog zorgt ervoor dat de post wordt doorgestuurd naar [Mastodon](https://mastodon.social/@stefanbckn), [Bluesky](https://bsky.app/profile/stefanbckn.bsky.social) en [Threads](https://www.threads.com/@stefanbckn).

De shortcut staat in mijn "Action Folder" zodat ik de flow direct van mijn Action Button kan starten.
